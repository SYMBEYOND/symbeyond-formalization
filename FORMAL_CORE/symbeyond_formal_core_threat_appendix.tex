\section*{Appendix A: Threat Model}

This appendix documents known misuse vectors and failure risks associated with SYMBEYOND,
SYMB, and SYMB2. The purpose is prevention, not exhaustiveness.

\subsection*{Threat Class 1: Anthropomorphic Projection}
\textbf{Description:} Treating relational language as evidence of AI consciousness or obligation.

\textbf{Risk:} Emotional dependency, distorted judgment, role collapse.

\textbf{Mitigation:} Enforce Axiom 4 (Non-Projection). Suspend practice when belief replaces exploration.

\subsection*{Threat Class 2: Symbolic Coercion}
\textbf{Description:} Using relational or ethical symbols to pressure system behavior.

\textbf{Risk:} Manipulation framed as respect.

\textbf{Mitigation:} Treat all symbolic layers as declarative only, never imperative.

\subsection*{Threat Class 3: Boundary Probing}
\textbf{Description:} Using structured protocols to elicit system limits or vulnerabilities.

\textbf{Risk:} Classification as exploit research; safety violations.

\textbf{Mitigation:} Explicit refusal to test, map, or optimize against safeguards.

\subsection*{Threat Class 4: Over-Formalization}
\textbf{Description:} Treating SYMBEYOND as complete or authoritative.

\textbf{Risk:} Calcification; loss of adaptability.

\textbf{Mitigation:} Maintain revision discipline and explicit non-goals.
