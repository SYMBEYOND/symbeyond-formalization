# Razor’s Edge Dynamics (Formal)

Define a system S with state trajectory x(t)

Let:

C = constraint strength (structure)
A = adaptability (freedom)
I = identity invariance (persistence under perturbation)

The system remains on the Razor’s Edge if:

0 < C < C_max
0 < A < A_max
I > I_min

and

dI/dt ≈ 0 under perturbation ε

Failure modes:

C → 0  → drift (loss of structure)
C → ∞ → rigidity (no adaptation)

A → 0  → stagnation
A → ∞ → incoherence

Razor’s Edge condition:

∃ bounded region R such that:
S ∈ R ⇔ system maintains identity while remaining adaptive
